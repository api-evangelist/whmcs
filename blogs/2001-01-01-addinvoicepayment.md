Adds payment to a given invoice. Request Parameters Parameter Type Description Required action string “AddInvoicePayment” Required invoiceid int Required transid string The unique transaction ID that should be applied to the payment. Required gateway string The gateway used, in system name format (for example, paypal or authorize ).
